# Release Notes - Lap v0.2.4

Lap v0.2.4 is a major usability and compatibility update. This release brings real Windows scanning optimizations with over 20% performance improvement, expands media support with EXR, PSD, HDR/RGBE, TGA, JPEG 2000, DDS, DPX, and QOI, and significantly improves daily file operations including drag-and-drop, copy/paste, move/copy workflows, album reordering, and external file handling.

This version also includes a broad polish pass across Lap's interface: cleaner dialogs, improved sidebars and panels, better keyboard navigation, more consistent controls, smoother layout behavior, and a more fluid product experience overall.

Special thanks to @Rockbard for reporting a large number of professional, detailed, and highly actionable issues that helped shape this release.

## 🚀 Highlights

- **Windows Scan Performance** — Optimized scan and file I/O paths with database connection pooling, merged file operations, scan summary logging, and more robust handling for disconnected external drives. Windows scanning is now more than 20% faster in real-world large-library workflows.
- **Broader Format Support** — Added EXR thumbnails and previews, plus additional image formats including PSD, HDR/RGBE, TGA, JPEG 2000, DDS, DPX, and QOI. Lap now documents support for 60+ photo, RAW, and video formats. (#147)
- **Better File Operations** — Improved move/copy workflows, added drag-and-drop to album folders with copy support, preserved original filenames during import, and added multi-platform clipboard copy/paste through native platform shims. (#153, #161, #162)
- **Smoother Album Management** — Albums can now be reordered directly with drag-and-drop, album roots can be dropped into, favorite folder counts are clearer, and root folders can be favorited or excluded from search. (#158, #168)
- **More Consistent Interface** — Standardized dialogs, refined sidebars and file info panels, improved status indicators, polished text contrast, and made modal, toolbar, preview, and selection controls feel more cohesive.
- **Keyboard and Selection Upgrades** — Added a Settings shortcuts tab, search panel shortcut, Ctrl/Cmd-click multi-selection, Ctrl+wheel thumbnail resizing on Windows, and Tab-key focus cycling between sidebar and content. (#117, #150, #154)
