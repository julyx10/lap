# Release Notes - Lap v0.3.0

Lap v0.3.0 is a major library-management and browsing update. It adds Smart Albums, persistent Collections, native Apple Live Photo support, a dedicated map view, and a four-pane comparison viewer. The release also makes large libraries faster to browse and search, while bringing a broad round of refinements to grouping, thumbnails, metadata, video, and selection workflows.

This version focuses on helping you find, review, compare, and organize more of your library without breaking the folder-first workflow that Lap is built around.

## 🚀 Highlights

- **Smart Albums** — Create rule-based albums and use built-in views such as Recently Added and On This Day. Custom Smart Albums support configurable grouping, sorting, ordering, and file-type filters, so saved views can match the way you work. (#29, #179)
- **Collections** — Create persistent Collections and drag files into them from anywhere in your library. Collections support reordering, counts, duplicate-aware drops, clearing, and removing files from a Collection without moving them to Trash.
- **Apple Live Photos** — Lap now recognizes HEIC/MOV Live Photo pairs, displays a LIVE badge, plays them in the viewer, and keeps linked MOV and AAE sidecars together during rename, move, copy, and delete operations. Companion videos stay out of normal file lists, and pairing is incrementally refreshed during scans. (#123)
- **Map and Location Browsing** — A new Map view shows a density heatmap of geotagged photos, lets you switch map styles, and provides a nearby-photos panel. Lap also reads GPS metadata from videos when available. (#171, #193)
- **Four-Pane Comparison** — Compare up to four images side by side in the image viewer, making culling and detailed review much easier. (#121)
- **Faster Large-Library Workflows** — Improved virtualized viewport loading, grouped browsing, semantic-search result loading, and similarity-search calculations reduce work when browsing or searching large libraries.
- **Flexible Library Browsing** — The new Library sidebar brings together all files, favorites, ratings, subjects, and date-based views. Automatic grouping adapts to the active view, including folders, dates, locations, cameras, and lenses.

## ✨ New Features and Improvements

- Added configurable media-information badges on thumbnails, including format, ISO, shutter speed, aperture, focal length, and exposure. (#174)
- Added AI PNG prompt import during scanning for Automatic1111, InvokeAI, ComfyUI, and NovelAI images; imported prompts fill empty comments without overwriting your own notes. (#197)
- Added file-type filters to AI Search and Smart Albums, and clearer visual grouping for filename and visual-search results.
- Added background options and a shortcut for the image viewer. (#173)
- Added hover playback for videos and Live Photos in file info, with improved Live Photo playback in the viewer.
- Added multi-file opening in an external application, including confirmation before opening more than 100 files. (#194)
- Added collection-scoped and selected-file-scoped duplicate scans, so deduplication can be targeted to the files you are reviewing.
- Added responsive Calendar columns and clearer grouped thumbnail views, breadcrumbs, sidebar sections, and preview badges.
- Added the current commit hash to the About dialog and a setting for automatic update checks. (#185)
- Refreshed app branding, navigation icons, thumbnail layout, Manage Libraries, and other sidebar and grid controls.

## 🐞 Bug Fixes

- Fixed duplicate-file creation during concurrent album scans. (#190)
- Fixed stale grouping and selection after moving or deleting files, including search-result filenames and visual groups.
- Fixed several Live Photo edge cases, including stale sidecar associations and incorrect sidebar counts.
- Fixed a crash caused by null virtual-scroll thumbnail placeholders. (#203)
- Fixed JPEG capture metadata recovery, lens-model priority, and inconsistent RAW metadata formatting. (#204)
- Improved video playback compatibility and fixed video preview zoom in the file-info panel. (#186)
- Fixed album-folder grouping and selection-restoration flicker. (#199)
- Fixed grouped dedup navigation, multi-select keyboard behavior, filmstrip layout, temporary-view counts, and library loading feedback.
- Improved selection and external-open safeguards for mixed or large selections.
