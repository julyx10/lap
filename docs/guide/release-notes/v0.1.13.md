# Release Notes - Lap v0.1.13

Lap v0.1.13 is a platform and usability update focused on image format support, desktop behavior on Linux, and smoother everyday library workflows.

## 🚀 Highlights

- Added support for **JXL (JPEG XL)** images.
- Added a new **mouse wheel control option** in the viewer, so you can choose whether the wheel changes images or zooms in and out.
- Replaced transient tooltips with a new **global toast system**, making action feedback more consistent across the app.
- Improved generated preview handling across image workflows for more consistent browsing and viewing.
- Added support for **renaming album root folders**.
- Improved library switching and **state persistence**, reducing unnecessary resets when moving between libraries.
- Refined file operation feedback and other interaction details across the app.
- Improved Linux desktop behavior, including titlebar and window decoration handling.
- Updated build and release workflows for better reliability across platforms.

## 🐞 Bug Fixes

- Fixed **Reveal in Finder/Explorer** so it correctly highlights the target file. (#51)
- Fixed Linux thumbnail and preview protocol URLs.
- Fixed several Linux window and titlebar issues to better align behavior with other desktop platforms.
- Improved library state handling during library switching.
- Improved Korean (`ko`) translation quality and consistency. (#55)
- Fixed illegible text on the website in light mode. (#54)
