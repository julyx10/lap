# Release Notes - Lap v0.1.11

Lap v0.1.11 is a workflow and format-support update focused on RAW browsing, external app integration, printing, and viewer polish.

## 🚀 Highlights

### 📷 RAW Photo Support
- **Major RAW formats supported:** Lap now supports browsing and indexing a wide range of mainstream RAW photo formats from Canon, Nikon, Sony, Fujifilm, Panasonic, Olympus, Pentax, Leica, Samsung, Kodak, Hasselblad, and more.
- **RAW thumbnails and preview:** Embedded previews are now used for fast RAW browsing, with format-specific fallback strategies for better compatibility.
- **RAW metadata and format labels:** RAW files now show normalized format labels and improved file information in the viewer and info panel.

### 🖨️ Printing
- **Built-in print preview window:** Images can now be printed through a dedicated print view instead of relying on external system commands.
- **System print dialog:** Lap now opens the system print dialog through the app WebView for a more consistent cross-platform workflow.
- **RAW and TIFF print preview:** RAW and TIFF images can be previewed in the print window through the same backend rendering path used for normal image preview.

### 🔗 External App Workflow
- **Open image or video in external apps:** Images and videos can now be opened directly in user-configured external applications.
- **Clearer context menu actions:** File actions were refined to distinguish image and video workflows more clearly.

### 🖼️ Viewer and Preview Improvements
- **Fullscreen refined in ImageViewer:** Fullscreen behavior is now focused on the dedicated ImageViewer window for a cleaner and more predictable experience.
- **Open in new window shortcuts:** `Shift + double-click` and `Cmd/Ctrl + double-click` now open items in a separate viewer window.
- **Improved preview behavior:** Preview, filmstrip, and related viewer toolbar behaviors were refined for consistency.

### ⌨️ Navigation and Selection
- **Keyboard navigation for album sidebar:** Album list and folder tree navigation now support arrow-key behavior closer to desktop file managers.
- **Better selection workflow:** In single-select mode, `Shift + click` now enters multi-select mode automatically and selects the full range.

### ⚙️ Settings and UI Polish
- **Improved settings structure and wording:** Settings sections, labels, and related i18n strings were reorganized for better clarity.
- **Window scaling refinements:** Settings and viewer windows now better follow the configured interface scale.
- **Update check behavior improved:** The About tab no longer checks for updates automatically on open, and manual update checks now have a timeout safeguard.

### 🧩 Fixes
- **Album file counter fix:** Moving files across albums now updates the source and destination album counters correctly in the UI.
